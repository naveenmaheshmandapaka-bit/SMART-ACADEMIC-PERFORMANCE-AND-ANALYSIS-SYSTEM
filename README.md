# 📊 Smart Academic Performance and Analytics System (SAPAS)

## 🚀 Project Overview

The **Smart Academic Performance and Analytics System (SAPAS)** is a Python-based application designed to manage and analyze student academic data efficiently. It allows users to store student details, track marks, perform basic analytics, and visualize performance using graphs.

This project demonstrates core programming concepts such as file handling, data structures, and data visualization in a simple and practical way.

---

## 🎯 Features

* ➕ Add student records
* 📋 View all students
* 🔍 Search student details
* ✏️ Update student marks
* 📊 Calculate average performance
* 📈 Visualize data using graphs
* 💾 Save and load data using CSV files

---

## 🛠️ Technologies Used

* Python
* CSV (File Handling)
* Matplotlib (Data Visualization)

---

## ▶️ How to Run

1. Install Python (if not already installed)

2. Install required library:

```
pip install matplotlib
```

3. Run the program:

```
python main.py
```
